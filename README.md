# color-picker
Um seletor de cores feito em JavaScript e CSS.
Ele permite que o usuário configure as cores nos formatos rgba, rgba, hexadecimal, hsl e hsla dinamicamente.
Cada configuração formada gera um código, que pode ser copiado e adicionado a uma folha de estilo.
## [Clique aqui para ver o seletor](https://jonatasmsantana.github.io/color-picker/)
